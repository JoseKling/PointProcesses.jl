Built-in models

Poisson processes

We provide a general PoissonProcess structure with a single intensity value and an arbitrary mark distribution.

julia> using MeasureTheory, PointProcesses, Random; Random.seed!(63)MersenneTwister(63)
julia> mark_dist = Dists.Categorical([0.1, 0.3, 0.6])Distributions.Categorical{Float64, Vector{Float64}}(support=Base.OneTo(3), p=[0.1, 0.3, 0.6])
julia> pp = PoissonProcess(5., mark_dist)PoissonProcess{Distributions.Categorical{Float64, Vector{Float64}}, Int64}(5.0, Distributions.Categorical{Float64, Vector{Float64}}(support=Base.OneTo(3), p=[0.1, 0.3, 0.6]))
julia> h = rand(pp, 0., 100.)TemporalHistory{Int64}([35.08442743828704, 64.41612667146856, 0.2252326463560994, 88.711215790212, 79.56689314287067, 93.0993103046172, 67.34815580817907, 98.97649185923194, 82.56376152900533, 24.63609595108618  …  16.92394480037176, 35.62561648689555, 99.3777147042952, 44.43452843504252, 88.78273000287686, 71.87464255040945, 62.775871815488095, 59.84396775405833, 23.619850940113917, 82.81728540702413], [3, 2, 3, 3, 2, 2, 2, 2, 3, 3  …  3, 2, 3, 3, 3, 2, 3, 2, 3, 1], 0.0, 100.0)
julia> pp_est = fit(PoissonProcess{Dists.Categorical}, h)PoissonProcess{Distributions.Categorical{Float64, Vector{Float64}}, Int64}(4.81, Distributions.Categorical{Float64, Vector{Float64}}(support=Base.OneTo(3), p=[0.1101871101871102, 0.30561330561330563, 0.5841995841995842]))
julia> intensity(pp_est)4.81
